## [1.0.0] - 2026-03-31

### Added
- Initial project setup with React frontend
- Express.js backend with API routes
- Visitor management system
- Access control and approval workflow
- Real-time alert system
- Admin dashboard
- Activity logging and audit trails
- Role-based access control (Admin, Guard, Resident)
- Mock data for development and testing
- Comprehensive API endpoints
- Authentication system with JWT
- Responsive UI with Lucide icons
- Component-based architecture

- ✨ Visitor entry registration and tracking
